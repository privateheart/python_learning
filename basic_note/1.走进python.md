## Python 标准数据类型

### 1.(长)整型

- 整型和长整型并不严格区分
- Python 2支持整型值后加 “L” 即为长整型
- 32位操作系统中，整型范围 -2^31 ~ 2^31-1
- 64位操作系统中，整型范围-2^63 ~ 2^63-1

### 2.浮点型

- 即数学中的实数

- 可以类似科学技术法表示

  ```python
  >>>3.22
  3.22
  >>>9.8e3
  9800.0
  >>>-4.78e-2
  -0.0478
  >>> type(-4.78e-2)
  <class 'float'>
  ```

### 3.复数型

- j = $\sqrt[]{-1}$，则 j 是 虚数

- 实数 + 虚数 就是复数

- 虚数部分必须有 j

  ```python
  >>>2.4+5.6j
  >>>type(2.4+5.6j)
  <class 'complex'>
  >>>3j
  3j
  >>>type(3j)
  <class 'complex'>
  >>> 5 + 0j
  (5+0j)
  >>> type(5+0j)
  <class 'complex'>
  ```

  

- 复数可以分离实数部分和虚数部分

  - 复数.real
  - 复数.imag

- 复数的共轭

  - 复数 .conjugate()

  ```python
  >>> # complex
  >>> x = 2.4+5.6j
  >>> x.imag
  5.6
  >>> x.real
  2.4
  >>> x.conjugate()
  (2.4-5.6j)
  ```
